XPUM_API xpum_result_t xpumGetDevicePowerMaxLimits(
    xpum_device_id_t deviceId,
    int* maxPowerLimits,
    bool* supported
    )

Get device max power limit.

This function is used to get the max power limit of device

Parameters:

deviceId

IN: The device Id

maxPowerLimits

IN/OUT: The detailed max power limit data. Parameter 'interval' has been obsoleted.

supported

OUT: support to get maxpowerlimit or not.

Returns:

xpum_result_t

  • XPUM_OK if query successfully

  • XPUM_GENERIC_ERROR if set failure
