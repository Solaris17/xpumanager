XPUM_API xpum_result_t xpum::xpumVersionInfo(
    xpum_version_info versionInfoList[],
    int* count
    )

This method is used to get XPUM version.

Parameters:

versionInfoList

IN/OUT: First pass NULL to query version info count. Then pass array with desired length to store version info data.

count

IN/OUT: When versionInfoList is NULL, count will be filled with the number of available version info, and return. When versionInfoList is not NULL, count denotes the length of versionInfoList, count should be equal to or larger than the number of available version info, when return, the count will store real number of entries returned by versionInfoList

Returns:

  • XPUM_OK if query successfully

  • XPUM_BUFFER_TOO_SMALL if count is smaller than the count of version info
